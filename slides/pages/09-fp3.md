# FP3 — Bouton, capture 1 s, WAV vers Audacity

**Pipeline d'enregistrement**

```mermaid {scale: 0.7}
flowchart TB
  BTN([Bouton D2]):::btn --> ARM
  ARM[ARMING<br>capture 8000 samples]:::st --> DUMP
  DUMP[DUMPING<br>16 008 octets binaires]:::st --> PY[Python<br>fp3_recv.py]:::pc
  PY --> WAV[.wav 8 kHz mono]:::pc --> AUD[Audacity]:::pc

  classDef st fill:#fef3c7,stroke:#d97706
  classDef btn fill:#fecaca,stroke:#dc2626
  classDef pc fill:#dbeafe,stroke:#2563eb
```

**Protocole série binaire** : magic header `0xAA55AA55` + uint32 length + 16 000 octets PCM int16 + footer `0xDEADBEEF`.

→ Logs ASCII suspendus pendant DUMPING.

::right::

**ET4 — preuve auditive**

À la lecture du `.wav` dans Audacity :

✓ on **reconnaît** « Électronique »<br>
✓ 4 syllabes distinctes visibles<br>
✓ bande utile préservée 0–4 kHz<br>
✓ pas de saturation, pas de clic
